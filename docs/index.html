SatiCode Configuration Schema

This site hosts the JSON Schema for SatiCode configuration files.

Configuration Schema

The main configuration schema is available at:

https://satibot.github.io/saticode/config.json

Usage

To use this schema in your SatiCode configuration file, add the following reference:

{
  "$schema": "https://satibot.github.io/saticode/config.json",
  "model": "MiniMax-M2.5",
  "autoupdate": true,
  // ... other configuration options
}

Schema Validation

This schema provides validation for:

  • AI model selection
  • Server configuration
  • RAG (Retrieval-Augmented Generation) settings
  • Provider configurations (MiniMax, OpenAI, etc.)
  • Tool configurations
